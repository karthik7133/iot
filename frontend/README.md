# Smart Irrigation Flutter App

A production-ready Flutter app for a Smart Adaptive Irrigation System.

## Features
- **Glassmorphism UI**: Premium dark mode design with blur effects.
- **Real-time Monitoring**: Periodic polling of sensor data from Render backend.
- **AI Insights**: Predictive analytics for soil moisture.
- **Water Savings**: Visualize efficiency gains.
- **Analytics**: Historical trends using `fl_chart`.

## Backend
The app connects to: `https://iot-0ts3.onrender.com`
